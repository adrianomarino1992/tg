O que são banco de dados

Bancos de dados ou bases de dados são conjuntos de arquivos relacionados entre si com registros sobre pessoas, lugares ou coisas. São coleções organizadas de dados que se relacionam de forma a criar algum sentido (Informação) e dar mais eficiência durante uma pesquisa ou estudo. São de vital importância para empresas e há duas décadas se tornaram a principal peça dos sistemas de informação. Normalmente existem por vários anos sem alterações em sua estrutura.

São operados pelos Sistemas Gerenciadores de Bancos de Dados (SGBD), que surgiram na década de 70. Antes destes, as aplicações usavam sistemas de arquivos do sistema operacional para armazenar suas informações. Na década de 80, a tecnologia de SGBD relacional passou a dominar o mercado, e atualmente utiliza-se praticamente apenas ela. Outro tipo notável é o SGBD Orientado a Objetos, para quando sua estrutura ou as aplicações que o utilizam mudam constantemente.

O que são banco de dados geograficos

Iniciamos nosso artigo falando sobre quem é referência no assunto, o INPE – Instituto Nacional de Pesquisas Espaciais, e como considera os principais conceitos neste assunto.

O primeiro termo que iremos definir é dado geográfico, suponhamos os seguintes exemplos:

Quais são as cidades vizinhas ao município de Capão Redondo?
A polícia precisando da seguinte informação: Quais ruas estão em um raio de um quilômetro em relação ao local onde aconteceu um furto?
Os extremos do país: Qual a distância entre Monte Caburaí- RO e Santa Vitória do Palmar- RS?

A resposta pra qualquer uma das perguntas contém informações geográficas específicas. Um banco de dados qualquer armazena informações sobre um endereço ou CEP, mas essa informação possui caráter secundário. Já para um dado como os exemplos citados é necessário o uso de dados geográficos. Tais dados possuem uma estrutura mais adequada para responder de maneira específica e até mostrar graficamente a resposta.

Aprofundando essa declaração, temos a definição de banco de dados geográficos, segundo Zoneamento Ecológico Econômico de Minas Gerais. Um Banco de Dados Geográfico é uma coleção de dados coerente e relacionável, suportando feições geométricas em suas tabelas.

O que é SQL

SQL é sigla inglesa de “Structured Query Language” que significa, em Português, Linguagem de Consulta Estruturada, uma linguagem padrão de gerenciamento de dados que interage com os principais bancos de dados baseados no modelo relacional.

Alguns dos principais sistemas que utilizam SQL são: MySQL, Oracle, Firebird, Microsoft Access, PostgreSQL (código aberto), HSQLDB (código aberto e escrito em Java).

SQL é uma linguagem essencialmente declarativa. Isso significa que o programador necessita apenas indicar qual o objetivo pretendido para que seja executado pelo SGBDR.
Alguns dos principais comandos SQL para manipulação de dados são: INSERT (inserção), SELECT (consulta), UPDATE (atualização), DELETE (exclusão). SQL possibilita ainda a criação de relações entre tabelas e o controle do acesso aos dados.

SELECT

A palavra reservada 'SELECT' é utilizada para descrevermos os campos que queremos de retorno no resultado.

Dentro de um SELECT podemos, alem de selecionar campos, tambem utilizar funções.

Exemplo : SELECT nome, idade, nota FROM aluno

Esta consulta irá retornar o nome, a idade e a nota de todos os alunos da tabela.







Exemplo : SELECT nome AS "aluno", nota AS "media" FROM aluno

Você tambem pode escolher um apelido para o retorno da sua consulta utilizando a palavra reservada 'AS'







Exemplo : SELECT 'Exemplo de SELECT' AS "O nome que quiser"

Você não precisa de uma tabela para realizar SELECTs



FROM

A palavra reservada 'FROM' é utilizada para descrevermos de onde virão nossos dados, de qual tabela.

Dentro do FROM informamamos todas as tabelas que iremos usar.

Exemplo : SELECT * FROM aluno

Esta consulta irá retornar todos os campos da tabela aluno.







Exemplo : SELECT * FROM aluno, materia LIMIT 1

Você pode consultar mais de uma tabela ao mesmo tempo, mas cuidado, o retorno será a combinação de todos resultados da primeira tabela x todos os resultados da segunda tabela.







WHERE

A palavra reservada 'WHERE' é utilizada para descrevermos filtros para seleção de dados.

Dentro do WHERE escrevemos condições que retornarão valores booleans(true/false).

Exemplo : SELECT aluno.nome as "aluno", materia.nome as "materia" FROM aluno,materia WHERE materia.nome = 'matematica' AND materia.aluno = aluno.rg

Esta consulta ira retornar o nome de todos os alunos que tiverem cursando a materia matematica







Exemplo : SELECT * FROM materia WHERE nome = 'matematica'

Esta consulta ira retornar os registros onde o nome da materia é matematica







Exemplo : SELECT * FROM aluno WHERE nome = 'adriano'

Esta consulta ira retornar os registros onde o nome seja adriano







MAX()

A função agrupadora MAX() é utilizada para obter o maior valor dentre todos os valores de um campo.

Exemplo : SELECT MAX(nota) FROM aluno

Esta consulta ira retornar a maior nota da tabela aluno



MIN()

A função agrupadora MIN() é utilizada para obter o menor valor dentre todos os valores de um campo.

Exemplo : SELECT MIN(nota) FROM aluno

Esta consulta ira retornar a menor nota da tabela aluno



AVG()

A função agrupadora AVG() é utilizada para obter a media dos valores de um campo.

Exemplo : SELECT AVG(nota) FROM aluno

Esta consulta ira retornar a media das nota da tabela aluno



SUM()

A função agrupadora SUM() é utilizada para obter a soma dos valores de um campo.

Exemplo : SELECT SUM(nota) FROM aluno

Esta consulta ira retornar a soma das nota da tabela aluno



GROUP BY

A palavra reservada 'GROUP BY' é utilizada para agrupar valores de acordo com algum campo da tabela.

Exemplo : SELECT materia.nome,COUNT(*) FROM materia GROUP BY materia.nome

Esta consulta ira retornar a quantidade de alunos cursando cada materia



ORDER BY

A palavra reservada 'ORDER BY' é utilizada para ordenar valores de algum campo da tabela.

Exemplo : SELECT materia.nome,COUNT(*) FROM materia GROUP BY materia.nome ORDER BY COUNT(*) DESC

Esta consulta ira retornar a quantidade de alunos cursando cada materia ordernados do maior para o menor



ST_AREA

A função ST_AREA é utilizada para obter o valor da area de uma geometria, caso esta seja do tipo Poligono.

Exemplo : SELECT nome,ST_AREA(geom) FROM cidade

Esta consulta ira retornar o nome e a area de cada cidade da tabela cidade



ST_TOUCHES

A função ST_TOUCHES é utilizada para obter valor boolean a partir de uma analize entre duas geometrias, caso estas geometrias sejam vizinhas, ou seja, tem um ponto de borda em comum, o retorno será true

Exemplo : SELECT a.nome FROM cidade as "a" , cidade as "b" WHERE ST_TOUCHES(a.geom,b.geom) AND b.nome = 'Jacareí'

Esta consulta ira retornar o nome das cidades vizinhas de Jacareí



ST_INTERSECTS

A função ST_INTERSECTS é utilizada para obter valor boolean a partir de uma analize entre duas geometrias, caso estas geometrias se cruzem, ou seja, tem um ponto em comum, o retorno será true

Exemplo : SELECT a.nome FROM microregiao as "a" , cidade as "b" WHERE ST_INTERSECTS(a.geom,b.geom) AND b.nome = 'Jacareí'

Esta consulta ira retornar o nome das estradas que passam por Jacareí



ST_BUFFER

A função ST_BUFFER é utlizada para gerar uma geometria de raio definido, a partir de uma outra geometria.

Atenção, o calculo é feito baseado no SRID da geometria selecionada

Exemplo : SELECT ST_BUFFER(ST_GEOMFROMTEXT('POINT(398113 7418092)',31983),100)

Esta consulta ira retornar uma geometria com raio de 100m a partir da geometria base.



ST_CONTAINS

A função ST_CONTAINS é utlizada para verificar se uma geometria comtem a outra em seu interior

Exemplo : SELECT nome FROM cidade WHERE ST_CONTAINS (geom,ST_GEOMFROMTEXT('POINT(398113 7418092)',31983));

Esta consulta ira retornar o nome da cidade em que o ponto está localizado;



ST_WITHIN

A função ST_WITHIN é utlizada para verificar se uma geometria esta totalmente dentro da outra

Exemplo : SELECT nome FROM cidade WHERE ST_WITHIN (ST_GEOMFROMTEXT('POINT(398113 7418092)',31983),geom);

Esta consulta ira retornar o nome da cidade em que o ponto está localizado, da mesma maneira que a ST_CONTAINS, mudando apenas a maneira como a buscar é feita;



ST_CENTROID

A função ST_CENTROID é utilizada para obter o centro de um poligono

Exemplo : SELECT ST_CENTROID(geom) FROM cidade WHERE nome ilike 'Jacareí'

Esta consulta ira retornar o centro da geometria da bacia do Paraguai



ST_GEOMFROMTEXT

A função ST_GEOMFROMTEXT é utilizada para transformar uma geometria descrita como texto em um objeto geometry

Exemplo : SELECT ST_GEOMFROMTEXT('POINT(398113 7418092)',31983);

Esta consulta ira retornar uma geometria do tipo point, no SRID 4326



ST_ASTEXT

A função ST_ASTEXT é utilizada para transformar uma geometria em uma geometria descrita como texto

Exemplo : SELECT ST_ASTEXT(geom) FROM cidade WHERE nome ilike 'Jacareí';

Esta consulta ira retornar a geometria da bacia escrita como um texto.
